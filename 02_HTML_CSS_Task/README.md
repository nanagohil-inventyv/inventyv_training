# Responsive Login & Signup Form

A simple and modern **Login / Signup web form** built using **HTML and CSS only**.  
The project focuses on responsive design, clean UI, and CSS-based interactions without JavaScript.

## Features
- 70 / 30 layout on desktop screens  
- Image on top and form below on mobile devices  
- Login / Signup toggle using pure CSS  
- Responsive design using Flexbox and media queries  
- Modern UI with gradients and shadows  

## Technologies Used
- HTML5  
- CSS3 (Flexbox, Media Queries, Responsive units)
